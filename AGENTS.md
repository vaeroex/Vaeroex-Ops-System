# Vaeroex Ops System - Agent Instructions

This project is a SaaS-style operations system for small and mid-sized businesses.

The product name is Vaeroex Ops System.

The in-app AI assistant name is Vaeroex.

Always refer to the assistant as Vaeroex in the user-facing UI.

The app helps businesses create forms, checklists, SOPs, tasks, reports, issue logs, asset tracking, workflow maps, and operations audits.

Do not overuse the word AI. This should feel like a practical operations platform.

Use clean TypeScript, reusable components, secure database access, and tenant-safe queries.

All workspace data must be separated by workspace_id.

Use Supabase Row Level Security policies where applicable.

Do not collect patient data, PHI, ePHI, Social Security numbers, medical record numbers, insurance IDs, or regulated healthcare data.

AI output should be treated as a draft or recommendation. Users must confirm before records are created or changed.

Prioritize practical business workflows over flashy UI.
